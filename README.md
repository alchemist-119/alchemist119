# 👋 Welcome｜歡迎

## 🇹🇼 中文

歡迎來到 **Alchemist 119**。

這裡不是一本科學課本。

也不是 AI 幫我完成作品的地方。

這裡記錄的是：

**一位學習者，如何透過提問、思考、修正，以及與 AI 的對話，慢慢建立自己的理解。**

每一篇漫畫，

都不是從答案開始。

而是從一句：

> **「為什麼？」**

開始。

漫畫只是成果。

真正珍貴的是：

那些還沒有答案時，

一次又一次的思考。

# ❤️ What We Believe｜我們相信

## 🇹🇼

知識，

可以搜尋。

答案，

可以查詢。

但是，

真正重要的是：

保持好奇。

勇於提問。

願意思考。

接受修正。

持續成長。

如果有一天，

有人因為看了這裡的一篇漫畫，

開始問出自己的第一個：

> **「為什麼？」**

那這個 Repository，

就完成了它的使命。

## 🇺🇸

Knowledge can be searched.

Facts can be memorized.

Answers can be found.

But curiosity,

critical thinking,

and the courage to revise ourselves

must be cultivated.

If someone leaves this repository

asking one more meaningful question

than before,

then it has already achieved its purpose.
