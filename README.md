這一次，我想挑戰三個看似毫無關聯的化學問題：

- 🟡 為什麼黃金比鐵軟？

如果把原子變成角色，

---

🏰 第一站：黃金 vs 鐵

硬度之謎

黃金為什麼比較軟？

不是因為它比較弱。

而是因為原子的排列方式，讓原子層更容易滑動，因此純金具有良好的延展性。

---

💍 第二站：K 金城堡

珠寶工匠加入銅或銀後，

原本整齊排列的黃金原子受到影響。

原子層變得不容易滑動，

戒指因此更加堅硬耐用。

這就是材料科學中的：

固溶強化（Solid Solution Strengthening）

---

⚡ 第三站：稀有氣體大反叛

大家都認為：

«稀有氣體永遠不會反應。»

直到超強氧化劑 PtF₆（六氟化鉑） 出現。

它成功把氙（Xe）的電子奪走，

第一次改寫了化學史。

科學家因此發現：

原來稀有氣體不是完全不反應，只是需要極端特殊的條件。
