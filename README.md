# 👋 Welcome

歡迎來到 **The Atomic Universe**。

這裡不是一本科學教科書，

也不是 AI 幫我完成的作品集。

這裡記錄的是——

> 一位學生如何提出問題、
> 如何與 AI 一起思考、
> 如何修正自己的理解，
> 最後把抽象的科學，
> 畫成每個人都能理解的漫畫。

---

# 💡 我相信

真正的學習，

不是背下更多答案。

而是一次又一次問：

> **為什麼？**

然後勇敢承認：

> **「我原本想錯了。」**

最後，

找到更接近真相的答案。

---

# 🧭 我如何學習？

每一篇漫畫，

都遵循同一個探索流程。

```text
提出問題
      │
      ▼
提出猜想
      │
      ▼
與 AI 討論
      │
      ▼
查閱資料
      │
      ▼
修正理解
      │
      ▼
再次推理
      │
      ▼
畫成漫畫
      │
      ▼
分享知識
```

> **Science is not about memorizing answers.**
>
> **Science is about learning how to think.**
