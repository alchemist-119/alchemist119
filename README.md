# 👋 關於 Alchemist 119

大家好，我是 **Alchemist 119**。

我是一位小六學生，喜歡科學，也喜歡畫漫畫。

我發現，真正讓我學會科學的，不是一次就答對。

而是一次又一次地：

**提問、猜想、試誤、修正，再重新理解。**

所以，我決定把這些思考過程畫成漫畫。

希望分享給同學、老師，以及每一位喜歡學習的人。

---

# 🌱 Alchemist 119 的學習方式

對我來說，

**Alchemist 119** 不是一個角色，

而是一種學習方式。

每一次遇到新的問題，

我都會努力完成這段旅程：

```text
❓ 提出問題
        │
        ▼
💭 自己先思考
        │
        ▼
🤖 與 AI 一起討論
        │
        ▼
❌ 發現自己的錯誤
        │
        ▼
📚 查資料、重新推理
        │
        ▼
🔄 修正自己的理解
        │
        ▼
🎨 畫成漫畫分享
```

AI 不會替我思考。

真正思考的人，

永遠是自己。
