# 👋 關於我

我是 **Alchemist 119**，一位熱愛科學的小六學生。

我喜歡問「為什麼」。

我也喜歡把自己的思考過程畫成漫畫。

這個 GitHub 並不是一本標準答案，也不是 AI 幫我完成的作品集。

它記錄的是我真實的學習旅程。

有時候，我會猜錯。

有時候，我會誤解一個科學概念。

有時候，我需要一次又一次修正自己的想法。

但正因為如此，我才慢慢理解：

> **真正的學習，不是一次就答對。**

> **而是在試誤中，一步一步接近真相。**

---

# 🌱 我希望分享的

我希望分享的，不只是科學知識。

更是一種學習的方法。

因為知識會一直更新，

但是勇於提問、願意修正、持續思考的能力，

會陪伴我們一輩子。

如果我的漫畫能讓一位同學開始多問一句「為什麼？」，

如果能讓一位老師願意鼓勵學生分享自己的猜想，

那我就覺得，這個 GitHub 已經完成了它最重要的使命。

---

# 💭 我的學習方式

每一篇漫畫，都是從一個真正的問題開始。

```text
❓ 提出問題
      ↓
💭 自己先思考與猜想
      ↓
🤖 和 AI 一起討論
      ↓
🔍 發現錯誤與盲點
      ↓
📚 查資料、重新推理
      ↓
💡 修正自己的理解
      ↓
🎨 畫成漫畫分享
```

> **AI 不替我思考。**

> **AI 陪我一起思考。**
